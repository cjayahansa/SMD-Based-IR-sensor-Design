# IR Sensor PCB (Altium Designer)

This project is a **single IR sensor PCB** designed using **Altium Designer**.  
It uses a **TCRT5000 IR sensor**, **LM393 comparator IC**, **SMD resistors (101, 102)**, and an **LED indicator**.

The **LM393** converts the IR sensor signal into a **digital output**, making it suitable for **line follower robots** and **object detection**.  
The PCB works with **3.3V and 5V** and can be connected to **Arduino or ESP32**.
